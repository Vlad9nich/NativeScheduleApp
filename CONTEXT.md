# Context

## Проект
Нативное Android-приложение на Kotlin (`NativeScheduleApp`), которое переносит функционал сайта kernelordexter.ru.

## Текущий этап работы (Багфиксы и улучшения)
1. **Название**: Изменено на "Расписание сессии ВГУИТ" в Manifest.
2. **Иконка**: Сгенерирована ИИ-иконка, субагенту поручено нарезать её в mipmap и подключить к приложению.
3. **Шрифты**: Субагенту поручено интегрировать шрифты Manrope и Oswald в Jetpack Compose Theme, чтобы UI соответствовал сайту.
4. **Загрузка расписания**: Логика парсинга расписания переписана под новый формат (`JsonWeekSchedule`).
5. **Карта Здания (ВГУИТ)**:
   - Извлекли картинки этажей из `data.unity3d` `base.apk` с помощью Python-скрипта (`UnityPy`).
   - Написали нативный UI карты на Jetpack Compose (`MapActivity.kt`) с поддержкой зума (pinch-to-zoom).
   - Написали алгоритм A* (`PathFinder.kt`) для роутинга маршрутов.
   - Сделали всплывающее диалоговое окно в расписании (переход из кабинета на карту).
   - *План на будущее*: Разметить JSON-граф реальных координат кабинетов по картинкам этажей.
